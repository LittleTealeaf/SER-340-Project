Assignment 3 - Find the Highest Balance

You are tasked with sifting through a large set of data, consisting of deposits for individuals. It is formatted as a text file, where each row is a number, representing the amount deposited (They will all be integers). Each person is separated by a blank line. For example: 

				1000
				2000
				3000

				4000
				5000
				6000

This indicates that one person has deposits of amounts 1000, 2000, and 3000, and another person has deposits of amounts 4000, 5000, 6000.

Your task is to find the largest amount that any one person deposited in total. The solution for the above example would be 4000 + 5000 + 6000 = 15000

You will find the data in the text file input.txt. You should return the answer to the standard output.

Submissions should be using the Rust language, with the file name solution.rs
